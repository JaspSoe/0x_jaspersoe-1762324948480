## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. Connect your wallet or sign in with X (Twitter)
2. Chat with SolanaBot about what you want to build
3. Bot generates and deploys your dApp instantly
4. Access your dApp dashboard to manage deployments
